A. ADDED componentDidMount() METHOD
-----------------------------------
Purpose: Initialize Dragula with three swimlane containers and set up event listeners

Code Added:
componentDidMount(){
    this.drake = Dragula([
      this.swimlanes.backlog.current,
      this.swimlanes.inProgress.current,
      this.swimlanes.complete.current
    ])

    this.drake.on('drop', (element, target) => {
      this.drake.cancel(true);
      this.handleCardMove(element, target);
    });
}

Key Points:
- Initializes Dragula with three swimlane container references
- Uses drake.cancel(true) to prevent React-Dragula DOM conflicts
- Sets up drop event listener to handle card movements

B. ADDED componentWillUnmount() METHOD
--------------------------------------
Purpose: Clean up Dragula instance when component is destroyed

Code Added:
componentWillUnmount() {

    this.drake.destroy();

}

Key Points:

C. ADDED handleCardMove() METHOD
--------------------------------
Purpose: Handle card movement logic when dropped in different swimlanes

Code Added:
handleCardMove(element, target) {
  if(!target) return;

  const cardID = element.getAttribute('data-id')

  let newStatus;
  if(target === this.swimlanes.backlog.current){
    newStatus = 'backlog'
  }
  else if(target === this.swimlanes.inProgress.current){
    newStatus = 'in-progress'
  }
  else if(target === this.swimlanes.complete.current){
    newStatus = 'complete'
  }
  
  this.updateCardStatus(cardID, newStatus);
}

Key Points:
- Extracts card ID from dragged DOM element
- Determines target swimlane by comparing with swimlane refs
- Maps swimlane to appropriate status string
- Calls updateCardStatus to handle state changes

D. ADDED updateCardStatus() METHOD
----------------------------------
Purpose: Update React state to move card between swimlanes

Code Added:
updateCardStatus(cardID, newStatus){
  const updatedClients = {...this.state.clients}

  let cardToMove = null;
  Object.keys(updatedClients).forEach((swimlanes) => {
    const cardIndex = updatedClients[swimlanes].findIndex( (client) => { 
      return client.id === cardID
    })
    if(cardIndex !== -1){
      cardToMove = updatedClients[swimlanes][cardIndex]
      updatedClients[swimlanes].splice(cardIndex, 1);
    }
  })

  if(cardToMove){
    cardToMove.status = newStatus;

    if(newStatus === 'backlog'){
      updatedClients.backlog.push(cardToMove)
    }
    else if(newStatus === 'in-progress'){
      updatedClients.inProgress.push(cardToMove)
    }
    else if(newStatus === 'complete'){
      updatedClients.complete.push(cardToMove)
    }
  }

  this.setState({ clients: updatedClients})
}

Key Points:
- Creates copy of current state to avoid mutations
- Finds and removes card from current swimlane array
- Updates card's status property to match new swimlane
- Adds card to appropriate target swimlane array
- Triggers React re-render with setState

TECHNICAL CHALLENGES RESOLVED
=============================

1. NODE.JS VERSION COMPATIBILITY ISSUE
---------------------------------------
Problem: Error: error:0308010C:digital envelope routines::unsupported
Root Cause: Project built for Node.js 11.10.0, running on Node.js v20.19.4
Solution: Used legacy OpenSSL provider command:
set NODE_OPTIONS=--openssl-legacy-provider && npm start

2. REACT-DRAGULA DOM CONFLICT
-----------------------------
Problem: NotFoundError: Failed to execute 'removeChild' on 'Node'
Root Cause: Dragula manipulating DOM elements while React trying to re-render
Solution: 
- Use drake.cancel(true) to prevent Dragula's DOM manipulation
- Let React handle all DOM updates through state changes
- Remove setTimeout delays that were causing timing issues

3. CODE QUALITY ISSUES FIXED
----------------------------
- Added missing quotes around 'backlog' status string
- Fixed missing return statement in findIndex callback
- Removed unused source parameter to eliminate warnings

IMPLEMENTATION FLOW
===================

Drag Operation Sequence:
1. User starts dragging a card (Dragula handles visual feedback)
2. User drops card in target swimlane
3. Dragula drop event fires
4. drake.cancel(true) immediately reverts DOM changes
5. handleCardMove() determines new status based on target
6. updateCardStatus() updates React state
7. React re-renders with card in new position and color

Status Mapping:
- Drop in backlog swimlane → status: 'backlog' → grey color
- Drop in inProgress swimlane → status: 'in-progress' → blue color
- Drop in complete swimlane → status: 'complete' → green color
